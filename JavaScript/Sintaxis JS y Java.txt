Secuncias de escape

\" = Mostrar una doble comilla simple dentro de una cadena en la salida de información.
\\ = Barra invertida, diagonal inversa, es para mostrar un back slash.
\n = Esto es un salto de línea.
\r = Retorno.
\t = Tabulación.
\b = Retroceso.
\f = Salto de página.
